# Oracle Protocol

Personal decision intelligence system. Track decisions, predictions,
and outcomes to build a calibrated model of your own cognition.

## Setup

1. Create a GitHub Personal Access Token with `repo` scope
2. Enable GitHub Pages for this repo (Settings > Pages > Source: main branch)
3. Visit `https://<username>.github.io/oracle-protocol/`
4. Enter your username, repo name, and PAT
5. Start logging decisions

## How It Works

- Log decisions BEFORE acting (prediction, probability, assumption, adversarial check)
- Revisit AFTER outcomes are clear (resolution, calibration, lesson)
- Dashboard shows your calibration curve and cognitive bias patterns

## Data

All decisions stored in `data/decisions.json`. Every save is a git commit.
Full decision history available through git log.
